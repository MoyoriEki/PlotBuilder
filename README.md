# PlotBuilder

漫画・物語のプロットをグリッドスナップ式ブロックとして配置・組み換えするWebアプリ「プロットボード」。

- 本体: `plotboard.html`(単一HTMLファイル・ノービルド)
- 公開URL: https://moyorieki.github.io/PlotBuilder/plotboard.html (GitHub Pages / mainへのpushで自動デプロイ)

## 設計方針

- 軽量・直感優先。D&DはPointer Eventsによる自前実装
- **縦長ファースト**: モバイル(スマホ/タブレット縦持ち)を主要ユースケースとし、初期グリッドは縦長(6列×12行)。今後のUIも縦長画面を優先して設計する
- データはIndexedDBへ自動保存(Phase 3でDropbox同期予定)
